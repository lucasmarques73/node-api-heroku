## Dependencies

- express
- supertest
- jest

## Install Dependencies

```shell
npm install
```

## Url Heroku

> https://node-api-heroku-lm.herokuapp.com/

## Running project

```shell
npm start
```

The project will be available in `http://localhost:3000`

## Running tests

```shell
npm test
```

## Running tests to generate covarage

```shell
npm run test:ci
```
